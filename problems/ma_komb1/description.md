### Kombinačné čísla
Vypočítajte paritu kombinačného čísla $n \choose k$.

#### Vstup
Na vstupe sú dve medzerou oddelené čísla $n$ a $k$ ($0 \leq k \leq n \leq 1\,000$).

#### Výstup
Ak je číslo $n \choose k$ párne, vypíšte `0`, inak vypíšte `1`.

##### Sample in
```
3 1
```

##### Sample out
```
1
```
